Бесплатный **генератор резюме** для Windows: создайте профессиональное резюме (CV) за минуты, выберите шаблон, заполните поля и сохраните **многостраничный PDF** формата A4. Без регистрации, без подписки, полностью офлайн.

> **Скачать готовую программу:** [Последний релиз (EXE)](https://github.com/ThodProd/o-sebe/releases/latest) — файл `О себе 1.0.0.exe`, portable, не требует установки.

---

## Возможности

- 6 шаблонов: Classic, Modern, Minimal, Dark, Elegant, White
- Экспорт в **PDF** (несколько листов A4 в одном файле)
- Редактор: опыт работы, образование, курсы, навыки, языки, фото
- Сохранение нескольких резюме на компьютере
- Автовосстановление черновика после закрытия
- Русский интерфейс

## Системные требования

- Windows 10 / 11 (64-bit)
- ~200 МБ свободного места

## Быстрый старт (пользователям)

1. Откройте [Releases](https://github.com/ThodProd/o-sebe/releases/latest)
2. Скачайте **`О себе 1.0.0.exe`**
3. Запустите файл (portable, установка не нужна)
4. Заполните резюме → кнопка **PDF** → сохраните файл

## Сборка из исходников

```bash
git clone https://github.com/ThodProd/o-sebe.git
cd o-sebe
npm install
npm run dist
```

Готовый EXE: `release/О себе 1.0.0.exe`

Разработка:

```bash
npm run dev          # Vite
npm run electron:dev # Electron + hot reload
```

## English

**O Sebe** is a free, open-source **resume and CV builder** for Windows. Create a polished A4 resume, pick a template, fill in your experience and education, and export a **multi-page PDF** — offline, no account required.

### Download

[**Latest release (portable EXE)**](https://github.com/ThodProd/o-sebe/releases/latest) — run without installation.

### Features

- 6 professional templates
- Multi-page A4 PDF export
- Local save for multiple resumes
- Photo upload and accent color per template
- Russian UI (content is yours — any language)
